目录结构及文件作用

了解和熟悉KingCMS目录结构，有利于进一步发掘更多的功能。

目录说明
[Root]网站根目录
[_cache]缓存目录,可以在config.php中指定
[api]API接口
uc.phpUCenter的API接口
[images]前台模板目录
[portal]自定义发布模型
[faq]自定义模型的常见问题
zh-cn.xml简体中文的常见问题
[language]语言包文件夹
zh-cn.xml简体中文语言包文件
_model.inc.php默认的模型表构成，可以删除
core.class.php自定义模型核心类
index.php前台用户发布功能页面
manage.content.php内容管理页面
manage.field.php字段管理页面
manage.model.php模型管理页面
manage.php管理首页
manage.tag.phpTag管理页面
shoppingcart.php购物车及相关处理页
tenpay.php财付通通知结果显示页
tenpay_remote.php财付通付款结果通知页
[system]主系统目录
[editor]编辑器目录
[nicEdit]nicEdit编辑器目录
[faq]主系统的常见问题
zh-cn.xml简体中文的常见问题
[images]系统所需图片
[lnk]快捷方式所需图片，图片大小必须为 32 x 32 px
err.gif错误提示图片
loading.gif加载过程中的16px图片
......
white.gif空的透明图片
[js]系统所需的javascript文件
jquery.jsjQuery核心文件
jquery.kc.jsKingCMS所需的jQuery扩展
[language]主系统所需的语言包
zh-cn.xml主系统的简体中文语言包
[lib]KingCMS类库
Cache.class.php缓存类
DateBase.class.php数据库操作类
KingCMS.class.phpKingCMS系统类
KingCMS.function.phpKingCMS基础函数
Language.class.phpXML语言包解析类
Skin.class.php管理系统的界面标签解析类
template.class.php模板解析类
verify.class.php验证码解析类
xml.class.phpXML文件解析类
[skins]管理界面皮肤目录
[default]默认皮肤文件夹，制作个性化的后台管理界面就靠这个了
default.htm默认html模板
system.home.htm管理首页的模板，这是后台皮肤制作里必须的
system.login.htm登录页的模板
ajax.pngajax界面图片
...图片部分省略
style.css管理界面样式表文件
[verify_font]验证码所需的字库目录
MilkCocoa.TTF英文字体文件，若要制作中文验证码，则删除此英文字库
拷贝中文字库到此目录下面，系统自己辨认
index.htm空页面
login.php管理登陆页
manage.php管理首页
verify.php显示验证码
[template]前台模板文件夹，仅是用来存储html模板，图片建议放到Root/images目录里
[inside]KingCMS特色的内部模板目录
[user]用户模块的动态页模板
index.htm用户中心页模板
login.htm用户登录页模板
...
[tag]Tag页面模板
default.htm默认的Tag页面模板
[article[list]]文章模型的列表页模板
[article[page]]文章模型的内容页模板
...
[bbs[list]]论坛模型的列表页模板
[bbs[page]]论坛模型的内容页，即帖子页模板
default.htm默认外部模板
home.htm首页外部模板
[user]用户模块
[client]UCenter的终端
[language]用户系统的语言包
core.class.php用户系统核心类
index.php用户中心
login.php用户登录页
manage.php用户管理中心
