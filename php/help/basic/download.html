获得KingCMS程序

从官方网站 http://www.kingcms.com/download/ 里下载得到。
除了官方网站直接可以下载得到外，还可以通过SVN中获取。

从SVN中获取

开发中的KingCMS程序是存储在GoogleCode代码仓库中，通过SVN工具可以获得最及时的更新代码。
建议使用ToroiseSVN，开发中的项目导出地址填写 http://kingcms.googlecode.com/svn/trunk/
稳定版本的项目导出地址为：http://kingcms.googlecode.com/svn/tags/

SVN目录结构说明

// trunk /			保存有当前最新的代码。但是主干分支并不是稳定的
// trunk / upload /		WEB程序文件，开发中的文件
// trunk / userguide /		用户使用指南

// tags /			冻结的代码库
// tags / tag_X.X.XXXX /	可用的代码，版本为X.X.XXXX

// branches /			对冻结的代码进行修改的时候，建立分支到此目录下面进行修改

// attachment /			附件
// attachment / png /		图片文件
// attachment / doc /		Word文件
